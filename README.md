# Bike Sales Data Analysis (Excel Project)

## Overview
This project analyzes bike sales data using Microsoft Excel to identify customer purchasing patterns and generate business insights through an interactive dashboard.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers
- Data Cleaning
- Dashboard

## Project Steps
- Cleaned and organized the dataset
- Created pivot tables to analyze customer demographics
- Analyzed sales based on:
  - Age
  - Gender
  - Income
  - Marital Status
  - Region
- Built an interactive dashboard using charts and slicers
- Generated insights to support business decision-making

## Dashboard Features
- Sales by Age Group
- Sales by Gender
- Sales by Income Level
- Customer Demographics Analysis
- Interactive Filters

## Key Insights
- Middle-aged customers purchase more bikes
- Higher-income customers are more likely to buy bikes
- Gender and marital status influence purchasing behavior

## Project File
Bike Sales.xlsx contains:
- Raw Data
